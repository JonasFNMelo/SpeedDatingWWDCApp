# SpeedDatingWWDCApp

Aplicativo iOS criado para facilitar uma dinâmica de **Speed Dating** durante o evento da **WWDC**, com o objetivo de quebrar o gelo e aproximar **Jr Learners** e **Alumnis**.

## Sobre o projeto

Eventos como a WWDC reúnem pessoas de turmas e experiências muito diferentes — de quem está começando a jornada no desenvolvimento iOS até quem já passou pelo programa e tem bagagem para compartilhar. Conectar essas pessoas de forma orgânica nem sempre é fácil, então a ideia foi adaptar o formato clássico de Speed Dating para esse contexto: rodadas curtas, perguntas pré-definidas e um cronômetro que mantém todo mundo no mesmo ritmo.

O app serve como o "mestre de cerimônias" da dinâmica:

- controla o **tempo de cada rodada**
- sugere **perguntas e tópicos** para destravar a conversa
- organiza as **transições entre rodadas**, deixando o foco totalmente na interação entre as pessoas

A proposta é simples: tirar a fricção do "e agora, sobre o que a gente fala?" e deixar que o encontro flua.

## Tecnologias

- **Swift**
- **SwiftUI**
- **Xcode**

## Como rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/JonasFNMelo/SpeedDatingWWDCApp.git
   ```
2. Abra o `SpeedDatingWWDCApp.xcodeproj` no **Xcode** (recomendado Xcode 15 ou superior).
3. Selecione um simulador de iPhone e execute (`⌘R`).
